新加坡投资移民

项目介绍

新加坡华人比例高达76%，生活习惯和思维模式保留中国传统，距离中国近，实施中英双语教育，非常适合中国申请人移民。新加坡是亚洲的避税天堂，资金安全，法制完善，犯罪率世界最低。新加坡税制低廉，海外收入无需申报纳税，无遗产税。入籍简单，获永居2年后即可申请入籍，护照免签170多国，含美加、英国、香港、日本以及大部分欧洲国家。

申请条件

  1. 公司成立超过3年，注册资本在100万人民币以上；
  2. 主申请人持股30%以上；
  3. 公司所属行业为房地产、建筑、汽车4s店以外的行业；
  4. 公司营业额符合移民局要求。

申请流程

  1. 2个月内收集资料递交到新加坡经济发展局；
  2. 2-4个月参加面试；
  3. 2-4个月获发原则性批复函；
  4. 1个月内完成投资；
  5. 1个月收到正式批复函；
  6. 2周完成体检并领取全家新加坡永居身份证。

所需费用

  1. 第三方费用：约2万元人民币（包含翻译费、公证费、审计费）
  2. 申请费：7000新币/户
  3. 体检费：新加坡体检55-120新币/人，国内体检约1700人民币/人
  4. 再入境许可费、身份证办理费：200新币/人
  5. 投资款：250万新币
