澳洲132商业天才移民

项目介绍

澳洲132签证又称“商业天才移民项目”，分为“显著商业成就”和“风险投资企业家”两大类。申请人需要在澳洲投资不少于150万澳币，且保持2年以上。132商业天才移民永居签证是目前澳大利亚所有投资移民中唯一可以一步到位获得永久居留权的签证，无需通过打分系统，对学历和英语无要求，对居住时间没有量化要求，适合那些有成功生意背景并且有意在澳洲做较大规模投资的企业家。申请人提交132签证之前要先提交EOI，而且要获得澳大利亚州政府或领地政府的担保。

申请条件

显著商业成就申请要求

  1. 主申请人年龄55周岁以下；
  2. 夫妻名下个人及家庭净资产不低于150万澳币；
  3. 过去4个财年中其中2个财年，拥有公司净资产不低于40万澳币；
  4. 过去4个财年中其中2个财年，公司年营业额不低于300万澳币；
  5. 过去4个财年中其中2个财年，夫妻名下持股比例不低于30%(营业额小于40万澳元，占股须51%以上；营业额为40万澳元以上，占股不低于30%)；如是上市公司则不低于10%。

风险投资企业家申请要求

  1. 从澳洲风险投资公司获得100万的资金；
  2. 该资金必须用于生意的早期启动、产品的商业化、或有前途的商业理念的生意发展；
  3. 风险投资公司必须为AVCAL中的一个成员，AVCAL网站上有会员风险投资类别的信息。

申请流程

  1. 申请人条件评估：确认能否申请；
  2. 申请州政府担保：每个州可能有不同要求；
  3. 获得签证邀请：1-3个月；
  4. 递交签证申请：1-2周；
  5. 补料信：按要求补交所需材料；
  6. 体检及补无犯罪，并且支付语言费；
  7. 获批132签证。

处理周期

9个月左右。

所需费用

  1. 州担保申请费：ACT堪培拉地区1100澳币，NSW新州境外申请1270澳币&境内1397澳币，QLD昆州750澳币，西澳750澳币，其他州不收取申请费；
  2. 签证申请费：主申6990澳币，年满18周岁申请人3495澳币/人，未满18周岁的申请人1745澳币/人；
  3. 体检费：每人1200人民币左右，外加250邮寄费（在澳洲移民局指定医院体检）；
  4. 语言费：雅思达不到平均4.5分需要交语言培训费，主申9795澳币，副申及18周岁以上的子女4890澳币/人，未满18周岁的子女无需交。
