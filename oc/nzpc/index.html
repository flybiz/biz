新西兰普通创业移民

项目介绍

新西兰普通创业移民适合于有过管理经验的申请人，要求申请人EOI分数达到120分，投资至少10万纽币，或者入股现有企业股份不低于25%。获得永居之后没有任何移民监的要求。世界上有171个国家对新西兰的护照免签，如美国、加拿大、澳洲等均可免签。

申请条件

  1. 拥有充足的管理经验（五年高管经验或三年持股25%以上股东经验均可）；
  2. 综合打分满足120分，并递交与管理经验相关的商业计划书；
  3. 在新西兰当地创业，投资至少10万纽币，并解释其合法来源，如果入股现有企业，持股不少于25%；
  4. 满足基本英语要求（雅思平均4分）。

申请流程

  1. 递交材料至新西兰移民局；
  2. 移民局审核客户资料，签发12个月工作签证；
  3. 持有工作签证在新西兰投资创业，12个月内申请24个月工作签证；
  4. 成功创业满2年申请居留权；
  5. 持有居留权满2年申请永久居留权。

处理周期

整体周期6-9个月获工作身份证，2年半-3年获居留权，4年半-5年获永久居留权。

所需费用

  1. 商业计划书费：5000纽币/户；
  2. 律师费：17000纽币/户；
  3. 签证申请费：3378纽币/户；
  4. 第三方费用：公证费、翻译费、房评费、审计费；
  5. 体检费： 成人约2000元人民币/人，小孩约1500元人民币/人（以体检医院收费标准为准）。
